# TransLine49ﺡﺍ monorepo

npm-workspaces monorepo for the TransLine49ﺡﺍ Environmental Services web presence.

```
apps/

  shared/            Reserved for cross-app code (empty for now)
```

## Develop

```bash
npm install
npm run dev
```

Visits `http://localhost:3000`.

## Build

```bash
npm run build
npm start
```
